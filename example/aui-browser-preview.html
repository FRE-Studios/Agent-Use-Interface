Tasks
Search Products
Search the product catalog. Use this when a user wants to find, browse, or discover products. The URL opens the search results page with filters pre-applied. On iOS and Android, this opens the native app search experience.
/search
search discovery
Parameters
q string required
The search query. Should reflect the user's intent in natural product-search terms (e.g., 'wireless earbuds noise cancelling').
example: wireless earbuds
category enum
Filter results to a specific product category.
electronics Phones, tablets, laptops, and other devices.
accessories Cases, cables, chargers, and add-ons.
audio Headphones, earbuds, speakers, and microphones.
wearables Smartwatches, fitness trackers, and AR glasses.
home Smart home devices, appliances, and IoT.
price_min number
Minimum price in USD. Use when the user specifies a budget floor.
min: 0
price_max number
Maximum price in USD. Use when the user specifies a budget ceiling.
min: 0
sort enum
How to sort results.
default: relevance
relevance Best match for the query (default).
price_asc Cheapest first. Use when user wants deals or is budget-conscious.
price_desc Most expensive first. Use when user wants premium options.
rating Highest rated first. Use when user wants the best-reviewed items.
newest Most recently added. Use when user wants the latest products.
in_stock boolean
Only show items currently in stock. Set to true unless the user explicitly wants to see out-of-stock items.
default: true
Examples
“Find me some good noise cancelling headphones under $200”
https://shop.example.com/search?q=noise+cancelling+headphones&category=audio&price_max=200&sort=rating&in_stock=true
“What's new in smart home devices?”
https://shop.example.com/search?q=smart+home&category=home&sort=newest&in_stock=true
Share Product
Generate a shareable link to a specific product. Use this when a user wants to send a product to someone or save it for later. The link opens the product detail page.
/product
share social
Parameters
sku string required
The product SKU. This must be a known product identifier. If the agent doesn't have the SKU, use the product-search task first.
pattern: ^[A-Z0-9\-]{6,20}$
example: WH-1000XM5
ref string
Attribution source. Set to 'agent' to indicate the link was constructed by an AI agent.
default: agent
utm_campaign string
Optional campaign tag for tracking.
Examples
“Send me a link to the Sony WH-1000XM5”
https://shop.example.com/product?sku=WH-1000XM5&ref=agent
Configure Wishlist View
Open the user's wishlist with specific display settings. Use this when a user wants to review their saved items in a particular order or filtered view.
/wishlist
personalization
Parameters
sort enum
How to sort wishlist items.
default: date_added
date_added Most recently saved first (default).
price_asc Cheapest first.
price_desc Most expensive first.
price_drop Largest price drop first. Use when user is bargain hunting.
notify_drops boolean
If true, enables price drop notifications for all wishlist items.
default: false
Examples
“Show me my wishlist sorted by biggest price drops”
https://shop.example.com/wishlist?sort=price_drop&notify_drops=true
